Configuration Notes
  • Values prefixed with @ are related field lookups — data is fetched at runtime via WebAPI from the entity referenced by the titleField lookup.
  • All other values are direct column bindings on the form entity itself.
  • Lookup columns (e.g., primarycontactid, msdyn_serviceaccount) automatically display the lookup display name.
  • OptionSet/Choice columns (e.g., statecode, industrycode) display the label text, not the numeric value.
  • The Compact layout uses section headers and key-value rows instead of icons and chips.
