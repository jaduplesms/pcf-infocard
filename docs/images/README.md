# README screenshots

Drop your captured screenshots here using these exact filenames so the
top-level `README.md` and `docs/*.html` pages pick them up automatically.

| File | What to capture |
|------|------------------|
| `layout-smart.png`   | A scenario with `layout = "smart"`   (e.g. *LIVE: Booking and Work Order Form*) |
| `layout-contact.png` | A scenario with `layout = "contact"` (e.g. *Contact Card*) |
| `layout-compact.png` | A scenario with `layout = "compact"` (e.g. *Customer Asset*) |

Suggested capture settings (PCF Workbench):

- **Device preset:** iPhone 14 (or any phone preset) — gives the card the
  same width it has in Field Service Mobile.
- **Theme:** Light. (A second dark-mode set with `-dark.png` suffixes is
  optional; the README only references the light variants today.)
- **Format:** PNG, ~520–600 px wide is plenty — the README renders them at 260 px.
- Crop to the control viewport (no Workbench chrome) for the cleanest look.
